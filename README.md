Unveiling the Android App Market: Analyzing Google Play Store Data

## Project Overview

This repository contains an analysis of Google Play Store data using Python. The dataset was cleaned, analyzed, and visualized to understand app categories, ratings, installs, pricing, and user reviews.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Dataset

- apps.csv
- user_reviews.csv

## Analysis

- Data Cleaning
- Category Analysis
- Rating Analysis
- App Size Analysis
- Install Analysis
- Free vs Paid App Analysis
- Sentiment Analysis
- Data Visualization

## Key Findings

- Family and Game categories contain the highest number of apps.
- Most apps are free.
- Most ratings are between 4.0 and 4.5.
- Highly rated apps generally receive more installs.
- Positive reviews are more common than negative reviews.

## Conclusion

The analysis helped identify app market trends and user preferences using Google Play Store data.
