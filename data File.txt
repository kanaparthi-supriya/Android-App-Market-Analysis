GOOGLE PLAY STORE DATA ANALYSIS

DATASET FILE INFORMATION

Dataset Source:
https://www.kaggle.com/datasets/lava18/google-play-store-apps

Files Used:
1. apps.csv
2. user_reviews.csv

Description:
The dataset contains information about Google Play Store applications and user reviews.

Features in apps.csv:
- App
- Category
- Rating
- Reviews
- Size
- Installs
- Type
- Price

Features in user_reviews.csv:
- App
- Translated_Review
- Sentiment
- Sentiment_Polarity
- Sentiment_Subjectivity

Tools Used:
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

Purpose:
To analyze app categories, ratings, installs, pricing trends, and user sentiments.
